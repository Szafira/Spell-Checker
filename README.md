## Android application name
Spell-checker

##General info
The application has been created for "Dungeon and Dragons" players for easier spell check.

## Technologies
This project was created with the use of:
-min SDK Version: 24
-target SDK Version: 30
-Java with the use of json
-D&D 5e API
